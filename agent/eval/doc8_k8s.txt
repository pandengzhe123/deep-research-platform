Kubernetes 是一个开源的容器编排平台，最初由 Google 设计并捐赠给云原生计算基金会（CNCF）。Kubernetes 于 2014 年首次发布，它的名字来自希腊语，意为"舵手"，常被简称为 K8s。Kubernetes 用于自动化容器化应用的部署、扩展和管理。Kubernetes 的核心组件包括控制平面和工作节点。控制平面包含 API Server、etcd、调度器和控制器管理器。etcd 是一个分布式键值存储，用于保存集群的所有配置数据。工作节点上运行 kubelet 和容器运行时。Kubernetes 中最小的调度单位是 Pod，一个 Pod 可以包含一个或多个容器。Kubernetes 通过 Service 提供稳定的网络访问，通过 Deployment 管理应用的副本和滚动更新。Kubernetes 支持水平自动扩缩容，可以根据 CPU 使用率或自定义指标自动调整 Pod 数量。Kubernetes 与 Docker 配合使用，Docker 负责容器化，Kubernetes 负责编排调度。Kubernetes 常用于微服务架构和云原生应用的部署。
