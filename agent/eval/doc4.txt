Linux is an open-source operating system kernel created by Linus Torvalds in 1991. Linux is the foundation of many operating systems called distributions, including Ubuntu, Debian, CentOS, and Arch Linux. The Linux kernel is written primarily in C and assembly language. Linux uses a monolithic kernel architecture and supports multitasking, multiuser operation, and symmetric multiprocessing. Android, the most popular mobile operating system, is based on the Linux kernel. Cloud computing platforms like AWS, Google Cloud, and Microsoft Azure heavily rely on Linux servers.
