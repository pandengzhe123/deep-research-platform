Git is a distributed version control system created by Linus Torvalds in 2005. Git tracks changes in source code during software development. Key Git concepts include commits, branches, merges, and remote repositories. Git is different from centralized version control systems because every developer has a full copy of the repository history locally. Popular Git hosting platforms include GitHub, GitLab, and Bitbucket. Git uses a content-addressable filesystem and stores data as snapshots rather than file differences.
