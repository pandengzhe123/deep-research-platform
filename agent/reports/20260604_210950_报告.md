# QQ飞车手游发展史研究报告

## 概述

《QQ飞车手游》是由腾讯天美工作室群开发的一款竞速类手机游戏，于2017年12月27日正式上线。该游戏基于2008年公测的《QQ飞车》端游IP开发，凭借对端游核心操作的高还原度、强大的社交系统以及持续的电竞化运营，迅速成为国民级竞速手游。上线首日DAU即突破1000万，上线一周DAU突破2000万，并将整个竞速品类市场规模提升了150%。截至2018年，手游累计注册用户突破2亿，端游与手游合计IP总用户超过7亿。游戏至今仍在持续运营，版本已迭代至1.56.0（2026年4月），并保持着高频的版本更新与IP联动活动。

## 核心发现

1. **端游IP的深厚积淀是手游成功的基石**：端游自2008年上线，积累了超过7亿IP用户，为手游提供了庞大的潜在用户池。手游立项时即设定“还原端游95%高阶操作”的目标，精准抓住了核心玩家的需求。
2. **上线即爆发，数据表现惊人**：手游上线24小时内DAU突破1000万，一周内突破2000万，首周注册用户达3500万，并曾同时占据App Store免费榜与畅销榜第一。上线首月月流水即超过1亿人民币。
3. **电竞化是长期运营的核心引擎**：2018年建立S联赛，吸引20支职业战队参与，2018年全年赛事观赛人次达4.8亿。2019年推出国际性赛事“亚洲杯”，持续提升游戏影响力和用户粘性。
4. **高频版本更新与跨界联动维持生命力**：游戏保持每年多次重大版本更新，并与敦煌研究院、兰博基尼、蜡笔小新、酷洛米等众多品牌和IP进行联动，不断注入新鲜内容。
5. **持续运营周期长，至今活跃**：截至2026年6月，游戏版本号已更新至1.56.0，S联赛春季赛总决赛仍在举办，显示出强大的长线运营能力。

## 详细分析

### 一、端游奠基与手游立项（2008-2017）

《QQ飞车》端游于2008年1月23日公测，灵感来源于《头文字D》，早期团队仅数人。上线首日最高同时在线（PCU）达15.9万，半年内日活跃用户（DAU）突破500万。2008年最高同时在线突破150万，注册用户达4.5亿。端游在2009年曾经历DAU下滑，但通过优化基础体验、增加社交系统（如2010年推出“师徒”系统）和赛事体系，于2010年8月14日使PCU突破100万，成为腾讯首款PCU超百万的自研产品。

手游立项始于2015年，由制作人郑磊带领4人团队预研，2016年底正式立项，目标为还原端游95%的高阶操作技巧。2017年4月20日，腾讯UP发布会正式宣布手游将于2017年上线，手游预约量最终超过4500万（另有来源称4700万），创下当时腾讯游戏预约纪录。

### 二、手游上线与爆发式增长（2017-2018）

《QQ飞车手游》于2017年12月27日正式上线，由腾讯天美工作室群开发。上线24小时内，DAU突破1000万；上线一周，DAU突破2000万，并曾同时占据App Store免费榜与畅销榜第一。上线首周注册用户达3500万。

手游的推出将整个竞速品类市场规模提升了150%，市场独占份额从37%升至90%。2018年1月，手游月流水即超过1亿人民币，常居App Store畅销榜TOP3。截至2018年，手游累计注册用户突破2亿，端游与手游合计IP总用户超过7亿。

### 三、电竞化发展与IP拓展（2018-至今）

**S联赛建立**：2018年5月26日，首届《QQ飞车手游》S联赛夏季赛正式打响，吸引了EDG、RNG、WE等20支职业战队参与。2018年全年赛事观赛人次达4.8亿，内容阅读量11.8亿次。

**国际化赛事**：2019年，赛事体系升级，推出首个国际性赛事——亚洲杯。第一届亚洲杯冠军由RNGM.云海获得。此后，赛事持续举办，2024年第六届冠军为RSG.风仔。

**IP联动与内容拓展**：游戏与多个知名品牌和IP进行联动，包括与敦煌研究院合作、小橘子担任英国旅游局友好大使，以及与兰博基尼、蓝翔、高智能方程式赛车、蜡笔小新、天线宝宝、酷洛米等展开联动。同时，飞车IP还通过漫画《飞车极速计划》和腾讯影业孵化的大电影进行内容深化。
